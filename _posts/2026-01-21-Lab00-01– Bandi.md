## Introduccion

OverTheWire Bandit es un wargame de dificultad progresiva enfocado en el aprendizaje práctico de Linux y fundamentos de seguridad informática. A lo largo de este desafío, el objetivo principal es obtener las credenciales necesarias para avanzar entre niveles, utilizando acceso remoto por SSH y herramientas básicas del sistema.

Durante el recorrido, nos enfrentaremos a distintos escenarios que requieren enumeración del sistema de archivos, manejo de permisos, análisis de archivos y uso de utilidades comunes de la línea de comandos. Cada nivel introduce un nuevo concepto que refuerza habilidades esenciales para el hacking ético, los CTFs y la administración de sistemas.

Bandit es un reto ideal para desarrollar una mentalidad metódica y fortalecer las bases técnicas que posteriormente se aplican en entornos más complejos de seguridad ofensiva


## Bandit – Nivel 0
Objetivo del nivel

El propósito de este nivel es conectarse por primera vez al servidor del juego utilizando SSH. Para ello, deberás establecer una conexión remota al host correspondiente, usando las credenciales iniciales proporcionadas por la plataforma.

Una vez que logres acceder correctamente al sistema, podrás continuar con el siguiente desafío consultando la página del Nivel 1, donde se explican las instrucciones para avanzar.

Este nivel sirve como introducción al entorno de Bandit y confirma que tienes acceso funcional al servidor remoto.


## Comandos útiles
SSH (Secure Shell) es un protocolo que permite conectarse de forma remota y segura a otro sistema a través de una red. Utiliza cifrado para proteger la comunicación, evitando que credenciales o datos sensibles sean interceptados.

En el contexto de Bandit, SSH es fundamental, ya que todo el wargame se juega mediante conexiones remotas a un servidor. Cada nivel requiere autenticarse por SSH con un usuario y contraseña específicos para poder avanzar.

Además de Bandit, SSH es ampliamente utilizado en:

Administración de servidores Linux

Pentesting y auditorías de seguridad

Transferencia segura de archivos

Automatización de tareas remotas

## Instalación de SSH según la distribución

## Debian / Ubuntu y derivadas
```javascript
❯ sudo apt update
sudo apt install openssh-client
```
## Arch Linux
```javascript
sudo pacman -S openssh
```
## Fedora
```javascript
sudo dnf install openssh-clients
```
## Bandit – Nivel 0 → Nivel 1

Para comenzar el Nivel 0, debemos conectarnos al servidor de Bandit utilizando SSH. El acceso se realiza con las credenciales iniciales proporcionadas por la plataforma.
```javascript
ssh bandit0@bandit.labs.overthewire.org -p 2220
```
Cuando el sistema solicite la contraseña, se debe ingresar:
```javascript
bandit0
```
